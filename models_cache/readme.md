# 🧠 Кэш ИИ-моделей (AI Models Cache)

В этой директории локально сохраняются скачанные веса и кэш нейросетевых моделей (HuggingFace, PyTorch, EasyOCR, YOLO..), которые используются для работы ИИ-агентов, распознавания текста и компьютерного зрения.

## ⚠️ Информация для разработчика

1. **Автоматическая загрузка:** При первом старте скриптов на новом железе Python автоматически скачает актуальные версии моделей из репозиториев и развернет их в этой папке. Вручную ничего переносить или скачивать не нужно.
2. **Исключение из контроля версий:** Сами подпапки с весами моделей (`/huggingface`, `/torch` и др.) жестко заблокированы в `.gitignore`. Они весят сотни мегабайт (и гигабайты), что ломает пуши на GitHub из-за лимита в 100 МБ.
3. **Зачем здесь этот файл:** Этот `readme.md` находится здесь исключительно для того, чтобы Git зафиксировал существование папки `models_cache/` в репозитории. При клонировании проекта на другую машину папка создастся пустой, что убережет скрипты от ошибок отсутствия путей (Directory NotFound).
